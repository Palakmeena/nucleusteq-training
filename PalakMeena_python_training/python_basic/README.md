## Overview
This folder contains solutions for the Python Basic Assignment. The exercises are organized into small, focused modules so each topic stays easy to understand, test, and explain.

The implementation follows Python best practices, including:

- PEP 8 coding standards
- Meaningful variable and function names
- Modular design
- Type hints
- Docstrings
- Exception handling where required
- Package organization
- OOP principles where they help keep code reusable

## Running Programs

Run any file directly from the repository root with Python:

```bash
python PalakMeena_python_training/python_basic/introduction/introduction_examples.py
python PalakMeena_python_training/python_basic/collections/data_structures.py
python PalakMeena_python_training/python_basic/exception/custom_errors.py
```

If you prefer running from inside a specific folder, navigate there first and execute the target file.

## Overview
This folder contains solutions for the Python Advanced Assignment. The implementation follows Python best practices, including:

- PEP 8 coding standards
- Meaningful variable and function names
- Modular design
- Type hints
- Docstrings
- Exception handling
- Package organization
- Object-oriented principles where applicable

The project is organized into separate modules based on the topics covered during training.

## Requirements

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running Programs

Navigate to the required folder and execute:

```bash
python file_name.py
```

Example:

```bash
python q41_two_threads.py
```

## Testing

Run all test cases:

```bash
pytest
```

Run a specific test:

```bash
pytest test_q32_addition.py
```
